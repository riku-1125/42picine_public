## Description
A collection of general-purpose C functions.
## Instructions
The following commands are available.
```
    make    : you can make all object files and libft.a.
    clean   : delete all object files.
    fclean  : delete all object files and libft.a.
    re      : remake all object files and libft.a.
```
## Resources
- man pages to check the function specifications.
- AI to validate the code and translation.

## Functions
- check a string(ft_isalnum,ft_isdigit and etc...)
- string manipulation(ft_strlen, ft_strtrim and etc...)
- memory manipulation(ft_memcmp, ft_memcpy etc...)
- list manipulation(ft_lstmap, ft_lstsize etc...)
